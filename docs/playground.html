OpenProof RPO Playground — narrative to open proof bundle

This page lets you test the open version of the RPO (Rapport Probatoire Ouvert): type a short narrative, add a few pieces of evidence, and generate an open RPO JSON skeleton.

CNRS-ready concept
This playground does not replace legal advice. It shows how an open proof standard can structure a case before any advanced AI analysis.

How this playground works

  1. You write a short narrative about a situation.
  2. You add 1–3 concrete evidence elements (dates, messages, events).
  3. The tool builds an open RPO JSON skeleton from your input.
  4. You can review / copy the JSON as a starting point for further tools.
If you want to test the scientific / CNRS pilot version with advanced psycho-forensic modules, you can apply here: TruthX Pilot Form.

Your mini-case

Start with a short, factual description. Then add a few concrete elements that could count as evidence (date, message, screenshot, witness, decision, etc.).

One paragraph is enough. Focus on events: who did what, when, and how it impacted you.
For now, this will generate a simple JSON skeleton. Later, it can be extended with AI assistance and PDF generation.
