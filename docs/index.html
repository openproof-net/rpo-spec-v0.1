From messy stories to verifiable bundles

RPO v0.1
is a proof bundle format,
not a storytelling machine.

RPO (Rapport Probatoire Ouvert) packages a case as one reusable bundle: signed JSON, a human-readable PDF path, and a public hash. Different institutions can verify the same file, without sharing private corpora.
