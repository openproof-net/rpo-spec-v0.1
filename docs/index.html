This page explains OpenProof RPO v0.1, an open standard for HR decision governance. It creates deterministic proof bundles for defensible HR decisions.

What you get in 30 seconds

Input 1–3 paragraphs
Output signed JSON + PDF + hash
Integrity anchor sha256
Purpose HR decision governance
RPO = signed JSON + human-readable PDF + public hash.
Deterministic, auditable, non-proprietary.
The hash proves integrity, not “truth”. It enables verification and accountability.
How it works
Step 1
Capture the decision context

A short narrative, with sources available at the time of decision. No data fishing, no retroactive reconstruction.

Step 2
Generate a deterministic bundle

Same input, same output: signed JSON, PDF rendering, and a public hash to guarantee integrity over time.

Step 3
Verify, audit, defend

Anyone can verify the bundle. Auditors can trace what existed, when, and under which responsibility perimeter.

About the author

OpenProof RPO v0.1 was created by Gersende Ryard de Parcey, architect of HR decision governance systems and founder of TruthX. Her work focuses on defensible HR decisions, data lineage, single source of truth architectures, and accountable decision infrastructures for regulated environments.
