From messy stories to verifiable bundles

RPO v0.1 is a proof bundle format, not a storytelling machine.

The RPO (Rapport Probatoire Ouvert) packages a case as one reusable bundle: signed JSON, a human-readable PDF path, and a public hash. The point is simple: different institutions can read, verify, and re-check the same file — without sharing private corpora.

Try the live demo See the mechanism

No AI. Deterministic pipeline: same input → same output. The sandbox shows the format. Explanation comes after the result.
