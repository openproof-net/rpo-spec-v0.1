RPO Test Suite

Minimal validation suite for RPO implementers: JSON schema validation, hashing integrity, and PDF narrative linkage.

1. Schema Validation Test

Validates candidate bundles against rpo-schema.json.

  • Ensures required fields exist
  • Checks types & constraints
  • Verifies structural coherence of the JSON bundle

View schema-validation.test.js →

2. Hash Integrity Test

Ensures that the canonical JSON produces a stable, verifiable hash.

  • Canonicalization process
  • SHA3-512 digest generation
  • Comparison with expected hash anchor

View hash-integrity.test.js →

3. PDF Output Test

Ensures that narrative PDF content corresponds to narrative_pdf_hash in the bundle.

  • PDF → text extraction
  • Canonical hashing
  • Matching with JSON field

View pdf-output.test.js →
