Decision defensibility simulator

Freeze the perimeter. Make decisions reconstructible.

This sandbox does not “choose” a candidate. It captures scope, declared systems, lineage hints and constraints, then generates deterministic outputs: sealed JSON + print view + public hash (SHA-256). No AI. No storage. Same inputs → same sealed core.

RPO = sealed JSON core + human-readable print view + public hash. The hash proves integrity over time, not “truth”. Verification is public. Accountability remains human.
